Anatomy of an Agent

soul.md — Who the agent IS
rules.md — What the agent MUST NOT do
tools.md — What the agent CAN use
Memory.md — What the agent HAS LEARNED
Four files. Human-readable, git-diffable, reviewable. Memory accumulates over time.
agents/pm/
  soul.md      # Personality & expertise
  rules.md     # Constraints
  tools.md     # Permissions
  Memory.md    # Persistent learnings

One Line to Join

# Install Rick
$ curl -fsSL https://raw.githubusercontent.com/Sagi363/Rick-POC/main/install.sh | bash

# Join a team universe
$ rick add git@github.com:Sagi363/Demo-Rick-Universe.git

# Compile agents
$ rick compile

Push It, Everyone Gets It

Update an agent — it's just a file change
rick push creates a branch + PR
Team pulls and recompiles
$ rick push
# ✓ Branch: rick/update-pm-agent-1234
# ✓ PR created

# Everyone else:
$ git pull && rick compile
